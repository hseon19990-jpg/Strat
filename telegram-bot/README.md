# بوت تيلغرام - SMMMAIN

بوت تيلغرام متكامل مرتبط بمنصة SMMMAIN.COM لإدارة خدمات السوشيال ميديا.

## المتغيرات المطلوبة في Railway

| المتغير | الوصف |
|---------|-------|
| `BOT_TOKEN` | توكن البوت من @BotFather |
| `OWNER_ID` | ايدي حساب تيلغرام المالك (رقمي) |
| `API_KEY` | مفتاح API من موقع SMMMAIN.COM |
| `ADMIN_GROUP_ID` | ايدي الكروب الذي تصله إشعارات الطلبات (رقمي، سالب) |

## النشر على Railway

1. ارفع الملفات على GitHub
2. اربط الـ repo بـ Railway
3. أضف المتغيرات أعلاه في قسم Variables
4. Railway سيشغل البوت تلقائياً

## أوامر المالك

| الأمر | الوصف |
|-------|-------|
| `/addpoints <user_id> <points>` | إضافة نقاط لمستخدم |
| `/broadcast <رسالة>` | إرسال رسالة لجميع المستخدمين |
| `/status <كود_الطلب>` | فحص حالة طلب |
| زر ⚙️ إعدادات المالك | لوحة التحكم الكاملة |

## إعداد البوت (للمالك)

### إضافة خدمة:
1. اضغط ⚙️ إعدادات المالك
2. اضغط ➕ إضافة خدمة
3. اختر الفئة
4. أرسل رقم الخدمة من موقع SMMMAIN
5. راجع المعلومات وأرسل الاسم بالعربية
6. أرسل: الحد الأدنى، الحد الأعلى، سعر النقطة

### الميزات:
- ✅ التحقق الرياضي عند الدخول
- ✅ نظام نقاط متكامل
- ✅ هدية يومية (قابلة للتعديل)
- ✅ نظام الدعوة برابط
- ✅ شحن النجوم (عدد نجوم أو عدد نقاط)
- ✅ استبدال النقاط بنجوم أو رقم تيلغرام
- ✅ تحويل النقاط بين المستخدمين (خصم 1%)
- ✅ تمويل القنوات (إجباري / داخلي)
- ✅ ربط كامل مع API موقع SMMMAIN.COM
- ✅ كود فريد لكل عملية
- ✅ إشعارات فورية للكروب
- ✅ إمكانية إلغاء أي صفقة وإعادة الرصيد
- ✅ إحصائيات البوت
- ✅ بث رسائل لجميع المستخدمين

## هيكل كود العملية

```
user_order_number - bot_user_number - total_bot_orders
مثال: 3-5-10
```
- الرقم الأول: رقم الطلب للمستخدم (طلبه الثالث)
- الرقم الثاني: رقم المستخدم في البوت (المستخدم الخامس)
- الرقم الثالث: الطلب العاشر إجمالاً في البوت
